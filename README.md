# Landing Page of Ambulink

## Getting started
1. install [node.js](https://nodejs.org/en/download)
1. Clone the repository 
    ```shell
    git clone https://github.com/AmbuLink-CS-55/landingpage.git
    ```
1. Go into the directory
    ```shell
    cd landingpage
    ```
1. install dependencies 
    ```shell
    npm install
    ```
1. Build the CSS 
    ```shell
    npm run build
    ```
1. Start development mode
    ```shell
    npm run watch
    ```
